# VERIFY 深度验证 Prompts — 妙笔小说

> 使用时直接复制对应模块的 prompt，粘贴给 `/verify` 指令。
> 每个 prompt 精确到具体点击路径和预期结果，不给模糊空间。
> 
> **新增方式**：在对应模块下新增 `### Vn. 场景名`，编号接续。新增模块在末尾追加。

---

## 大纲页验证

### V1. 大纲节点点击响应
```
打开APP → 进入大纲页 → 
依次点击：全书大纲 → 分卷大纲 → 全书大纲下的第1个大节点 → 
大节点下的第1个小节点 → 分卷大纲下的第1个卷 → 卷下的第1章 →
每次点击后确认右侧编辑器标题栏更新为当前选中节点名称 →
截图保存到 test_screenshots/outline_node_click.png
```

### V2. 加子节点功能
```
打开APP → 进入大纲页 → 点击"全书大纲" → 点击工具栏"+子"按钮 →
确认出现新的大节点（命名如"大节点N"）且自动选中 →
点击新大节点 → 点击"+子" → 确认出现新的小节点 →
截图保存到 test_screenshots/outline_add_child.png
```

### V3. 全书大纲/分卷大纲无"＋同"按钮
```
打开APP → 进入大纲页 → 点击"全书大纲"节点 →
确认工具栏中只有"+子"按钮，无"＋同"按钮 →
点击"分卷大纲"节点 → 同样确认无"＋同"按钮 →
点击普通大节点 → 确认"＋同"按钮出现 →
截图保存到 test_screenshots/outline_no_sibling.png
```

### V4. 大节点3加小节点
```
打开APP → 进入大纲页 → 展开"全书大纲" → 找到第3个大节点（索引0/2）→
点击选中 → 点击"+子" → 确认新的小节点出现在大节点3下方 →
点击新小节点 → 确认右侧编辑器可编辑 →
截图保存到 test_screenshots/outline_node3_child.png
```

### V5. 编辑器保存
```
打开APP → 进入大纲页 → 点击任意节点 → 在编辑器中输入"测试内容ABC" →
点击保存 → 切换到其他节点 → 再切回刚才的节点 →
确认编辑器显示"测试内容ABC"（内容已保留）→
截图保存到 test_screenshots/outline_save.png
```

### V10. 章节切换与内容加载
